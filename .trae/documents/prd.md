## 1. 产品概述

AI穿搭助手是一款面向时尚爱好者的智能穿搭推荐应用，通过AI算法结合天气、场景、个人风格等多维度数据，为用户提供个性化的穿搭方案与潮流资讯。
- 解决用户日常穿搭决策困难的问题，降低搭配门槛，提升穿搭品质
- 面向18-35岁追求时尚与品质生活的都市人群，市场价值在于AI+时尚的差异化定位

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 手机号/邮箱注册 | 浏览穿搭推荐、查看资讯、个人中心管理 |
| VIP用户 | 付费订阅 | AI穿搭效果图生成、专属风格分析、优先推荐 |

### 2.2 功能模块

1. **登录页**：品牌视觉展示、账号登录/注册、第三方登录入口
2. **首页**：天气动态卡片、今日穿搭推荐、场景快捷入口、潮流速递
3. **智能穿搭推荐页**：多维度筛选、穿搭卡片列表、搭配详情展开
4. **AI穿搭效果图页**：AI生成穿搭效果图、风格切换、保存/分享
5. **个人中心**：用户信息、衣橱管理、穿搭收藏、风格偏好设置、主题切换
6. **潮流资讯页**：资讯瀑布流、分类标签、热门话题、详情阅读

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：温柔玫瑰粉 #E8A0BF / 柔和薰衣草紫 #C3ACD0 / 奶油白 #F7F5F2
- **辅助色**：暖灰 #9B8EA8 / 深棕 #5C4033 / 金色点缀 #D4A574
- **暗色主题**：深紫灰 #1A1625 / 暗玫瑰 #8B5E83 / 柔光紫 #6B5B8A
- **按钮风格**：大圆角(16px)、渐变填充、hover缩放+阴影增强
- **字体**：展示字体 Playfair Display / 正文字体 Noto Sans SC
- **布局风格**：卡片式布局、底部导航栏(移动端)、侧边导航(PC端)
- **图标风格**：线性图标(lucide-react)、2px描边、圆角端点

### 4.3 响应式设计

- **桌面优先**：最大宽度1440px居中，侧边导航+内容区布局
- **平板适配**：768px-1024px，侧边导航收起为底部导航
- **手机适配**：<768px，底部Tab导航、单列卡片、触摸优化(最小点击区域44px)
- **断点**：sm(640px) / md(768px) / lg(1024px) / xl(1280px)

### 4.4 主题系统

- 浅色主题：奶油白底+玫瑰粉点缀+暖灰文字
- 深色主题：深紫灰底+暗玫瑰点缀+浅紫灰文字
- 主题切换：Zustand全局状态管理、CSS变量动态切换、过渡动画
